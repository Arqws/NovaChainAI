# RAG (Retrieval-Augmented Generation)

NovaChain AI uses a strict RAG approach to ensure answers are grounded only in internal documents.

## Why RAG?

- Prevents hallucinations
- Ensures data privacy
- Suitable for enterprise & NDA environments

## RAG Pipeline

1. Document upload
2. Text extraction
3. Chunking
4. Local embeddings
5. Vector search
6. Context-aware answer
